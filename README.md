# Portfolio Website

Personal portfolio website built with React and Vite.

## Development

```bash
npm install
npm run dev
```

## Deployment

This site is automatically deployed to GitHub Pages using GitHub Actions. The workflow triggers on pushes to the `main` branch.

### Manual Deployment Steps

1. Ensure your repository is connected to GitHub
2. Push your code to the `main` branch
3. Go to repository Settings â†’ Pages
4. Under "Source", select "GitHub Actions"
5. The site will be available at: `https://username.github.io/repository-name/`

Note: If your repository is named `username.github.io`, the site will be available at `https://username.github.io/` (root domain).
